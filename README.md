## üìñ About the Project

**TechBank 2.0** is a modern, responsive web application simulating a digital banking interface. 
It provides a dynamic user experience with features like real-time transaction updates, balance visibility toggling, and a fully functional dark mode, designed to work seamlessly across mobile, tablet, and desktop devices.

## üöÄ Features

* **Financial Dashboard:** Displays current balance and total income.
* **Dynamic Transactions:** Add new entries/exits via a custom modal interface, updating the list and balance automatically.
* **Dark Mode Integration:** Seamless theme switching using CSS variables.
* **Privacy Mode:** Hide or reveal sensitive balance information.
* **Responsive Design:** Built with CSS Grid and Flexbox for a flawless layout on any screen size.
